Hexo+Github 小白完全搭建指南

为什么要搭建一个个人博客

很久以前就想建一个自己的博客，之前以为在网易等等国内平台上注册一个账号，像QQ空间一样往里面写一点东西就算是一个博客了，但渐渐发现这样限制太大，直到最近才发现可以用 Hexo和GitHub搭建起自己的真正拥有所有权限的博客，于是在经历了无数次的失败尝试后终于成功。看着网上的教程，普遍不够完整，于是决定自己写一篇，也用来当做第一次使用这个博客的纪念。

安装hexo

1.打开 开始 -> Git Bash

2.输入
　npm install -g hexo

3.等待安装完成

4.打开D盘或是E盘：
　1）若安装了cmder，则右键，打开cmder
　２)若未安装，则 shift+右键，此处打开命令窗口

5.输入
hexo init blog
提示成功　INFO Start blogging with Hexo!

6.因为创建的文件夹里自带一篇hello world的文档，所以现在可以小试一下我们刚刚搭起来的博客了
输入 

hexo generate
hexo server
然后在浏览器输入 http://localhost:4000/ ![](http://i.imgur.com/QXtCFnU.jpg) 然后就能看到这样的图片，我们网站的初步模样。 --- # 配置GitHub 首先注册、登录 https://github.com/ 记住自己的Username（很重要） 然后右上角选择 Create a new repository https://github.com/new Repository name （填自己的名字） yourname.github.io(yourname与你的注册用户名一致,这个就是你博客的域名了) 例如，我的域名是github.com/JnuYHui，就填入JnuYHui.github.io --- # 把网页发布到GitHub 重新打开CMD，输入： ssh-keygen -t rsa -C "Github的注册邮箱地址" 一路Enter过来就好，得到信息： Your public key has been saved in /c/Users/user/.ssh/id_rsa.pub. 找到文件 id_rsa.pub 用（sublime text）打开，Ctrl + a复制里面的所有内容，然后进入Sign in to GitHub： https://github.com/settings/ssh New SSH key ——>Title：blog —— Key：输入刚才复制的 ——> Add SSH key --- # 博客的基本配置 在blog目录下，用sublime打开_config.yml文件，修改参数信息 　　**特别提醒，在每个参数的：后都要加一个空格** 修改网站相关信息 
title: 宇辉的博客 
subtitle: 副标题
description: 网页描述
author: 宇辉
language: zh-CN
timezone: Asia/Shanghai
配置部署（我的是jnuyhui，修改成自己的） 
deploy: 
  type: git
  repo: https://github.com/jnuyhui/jnuyhui.github.io.git
  branch: master

发表文章

在cmder中输入 

hexo new "第一篇博客"
在blog/source/_post文件夹中找到该文章，使用MarkdownPad打开 
---
title: 第一篇博客
date: 2017-03-04 13:03:44
tags:
---
这是一篇测试文章，欢迎关注作者博客: https://JnuYHui.github.io/
保存 在blog文件夹中打开cmder，依次输入以下命令 `hexo clean ` `hexo generate` `hexo server` 这时打开http://localhost:4000/ 可以发现，这篇文章已经发布成功 --- # 发布到网上 在blog文件夹中打开cmder，执行 
hexo deploy
会跳出Github登录，直接登录 如果没有问题输入(yourname).github.io/ 然后就可以看到已经发布了 --- # hexo常用命令 Hexo常用的几个命令： 
hexo generate (hexo g) 生成静态文件，会在当前目录下生成一个新的叫做public的文件夹
hexo server (hexo s) 启动本地web服务，用于博客的预览
hexo deploy (hexo d) 部署播客到远端（比如github, heroku等平台）
另外还有其他几个常用命令： 
hexo new "postName" #新建文章
hexo new page "pageName" #新建页面
常用简写 
hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy
常用组合 
hexo d -g #生成部署
hexo s -g #生成预览
